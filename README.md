# 簡易記事本App

<h3>功能介紹</h3>
<P>● 點擊主畫面右下角的"+"，會跳轉至添加記事頁面，輸入標題及內容後可添加記事。</P>
<P>● 點擊主畫面右上角的尋按鈕後，輸入關鍵字可搜尋相關的記事。</P>
<P>● 在某個記事上長按，可選擇刪除記事或編輯記事。</P>
<P>● 點擊主畫面右上角的選單，可選擇主畫面的顯示方式(列表顯示、網格顯示)。</P>
